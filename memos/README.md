# Memos

The deliverable. One synthesized memo per debate, named `YYYY-MM-DD-<topic>.md`, written by the CEO from [TEMPLATE.md](TEMPLATE.md).

Each memo has:
- a **Short summary** (3 sentences: recommendation / core tension / next action) — also the audio script if you use the optional audio skill;
- a **Long summary** — closing positions per persona, tensions resolved + unresolved, what would change the recommendation, action items by week, success criteria, and a revisit condition.

A weak brief produces a `-sharpen.md` memo instead (status `needs-sharpening`) — that's the gate telling you to sharpen the brief before any debate runs.

`audio/` holds optional MP3s (gitignored by default). A worked example memo is at [EXAMPLE-ship-now-vs-polish.md](EXAMPLE-ship-now-vs-polish.md).
