# Conversations

Full debate transcripts, one per debate (`YYYY-MM-DD-<topic>.md`), plus the per-round shared scratchpads (`scratch-round-2-context.md`, etc.) that show exactly what context each persona saw.

Written continuously as the debate runs — every persona's Round 1 / Round 2 / final position, verbatim, tagged with model + persona (e.g. `[Sonnet-4.6 · Distributor]`).

**Gitignored by default** — debate transcripts can contain rough thinking-in-progress. If you want full audit history tracked in git, delete the matching lines in [../.gitignore](../.gitignore).

This is the audit trail: for when you want to re-read the actual argument, not just the synthesis.
