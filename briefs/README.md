# Briefs

Your inputs. One markdown file per decision, named `YYYY-MM-DD-<topic>.md`.

- Copy [TEMPLATE.md](TEMPLATE.md) to start, **or** just describe your decision in chat and let the agent interview you and draft it.
- The CEO runs a 7-criterion quality gate on every brief *before* debating. A weak brief gets a `-sharpen.md` memo instead of a debate — fix what it names and re-run.
- A complete worked example lives at [EXAMPLE-ship-now-vs-polish.md](EXAMPLE-ship-now-vs-polish.md). Read it alongside its memo in `../memos/` to see the full input → output.

See [../HOW-TO-USE.md](../HOW-TO-USE.md) for the field-by-field guide.
